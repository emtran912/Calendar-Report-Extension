Setup notes

Before loading the extension, open Google Cloud Console, enable the Google Calendar API, and create an OAuth client for a Chrome extension.

  1. Load the extension unpacked once to get the extension ID.
  2. In Google Cloud Console, create OAuth credentials for a Chrome Extension and use that extension ID.
  3. Add the OAuth client ID back into manifest.json under oauth2.client_id.
  4. Keep the scope as https://www.googleapis.com/auth/calendar.readonly.

Add this field to the manifest after you create the OAuth client:

"oauth2": {
  "client_id": "YOUR_CLIENT_ID.apps.googleusercontent.com",
  "scopes": ["https://www.googleapis.com/auth/calendar.readonly"]
}

You can then generate reports manually in the popup or let the background alarm refresh a weekly snapshot.
